\## How to contribute



1\. Fork this repository.  

2\. Clone your fork to your local computer.  

3\. Make your changes (code, docs, bug-fixes).  

4\. Test your changes by running the pipeline or tests.  

5\. Commit your changes with a clear message.  

6\. Push to your fork and open a Pull Request.



\## Coding style



\- Add clear comments in scripts  

\- Follow the existing structure (scripts/, docs/, etc.)  

\- If you add new dependencies, list them in `packages.R`
